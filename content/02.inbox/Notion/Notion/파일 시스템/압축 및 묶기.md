```
압축
xz : xz -d 파일명.xz (많이 사용)
bzip2 : bzip -d 파일명.bz2
bunzip2 : 위와 동일
gzip : gzip -d 파일명.gz(매우 많이 사용)
gunzip : 위와 동일
```

```
묶기
tar
동작 : c(묶기) x(풀기) t(경로확인)
옵션 : f(파일) v(과정 보이기) J(tar+xz) z(tar+gzip) j(tar+bzip2)

예시
tar xvzf 파일명.tar.gz
tar xvjf 파일명.tar.bz2
```
