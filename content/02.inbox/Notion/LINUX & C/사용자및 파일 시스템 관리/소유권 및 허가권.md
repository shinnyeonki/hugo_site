허가권 변경

chmod

  

소유권 변경

chown 사용자 소유권

chgrp 그룹 소유권

  

default 허가권 변경

umask

리눅스의 기본 허가권은 파일의 경우 666

디렉토리의 경우 777 인데 여기서 umask 022 로 설정하면

기본 허가권이 파일은 644 디렉토리의 경우 755로 바뀐다
