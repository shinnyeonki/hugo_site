리눅스의 파일시스템은 3가지의 타임스탬프를 관리한다

```
stat : 파일시스템 명령어
* access time : atime (stat -c "%x")
* modification time : mtime (stat -c "%y")
* change time : ctime (stat -c "%z")
```
