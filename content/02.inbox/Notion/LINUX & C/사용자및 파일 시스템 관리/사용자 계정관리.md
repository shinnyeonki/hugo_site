```
관련 설정 파일
/etc/passwd 유저 정보 포함
/etc/shadow 패스워드 암호화 및 연관 속성
/etc/default/useradd 유저생성시 기본 설정 값
/etc/login.def 파일을 참조하여 변수에 따라 생성
```

  

  

```
유저추가 삭제

adduser : 고수준 : redhat 은 차이 없음
useradd :  저수준
passwd : 사용자 비밀번호 변경
usermod : 사용자 설정변경
userdel : 사용자 삭제
chage : 사용자 패스워드 관리

groupadd : 그릅 생성
groupmod : 그룹정보 변경
groupdel 그룹 삭제
gpasswd : 그룹 비밀번호 변경
newgrp : 새 그룹 참여하기
```
