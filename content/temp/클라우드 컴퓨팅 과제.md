## 하둡 컨테이너를 구성하고자 하는 환경(가상머신, 리눅스 서버 등) 설명
초기 arm 기반 mac 에서 실행되지 않아
홈서버를 사용하여 x86-64 cpu, linux ubuntu server 버전에서 ssh 로 접근하여 사용하였음
```bash
-p 50070:50070 
-p 50030:50030
``` 
등을 사용하여 실행시 적절하게 외부에서 hdfs mapreduce 웹 인터페이스 또한 접근가능
