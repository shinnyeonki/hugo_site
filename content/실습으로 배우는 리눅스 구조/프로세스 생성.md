프로세스는 두가지 목적으로 생성한다
- fork() : 다중 프로세스 프로그램 사용 => 현재 프로세스의 메모리 복사 다른 프로세스로 등록
- execve() : 다른 프로그램 실행용( bash 에서 실행할 때 ) => 현재 프로세스의 메모리를 새로운 프로세스의 메모리로 덮어 쓴다

실행파일은 리눅스에서는 ELF (excutable linkable format) 형식이며

readelf -S {실행파일 명}
